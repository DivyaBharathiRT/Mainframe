

	MOVE TI001-ID TO HV-ID
      EXEC SQL
		SELECT 
			ID
			,NAME
			,SAL
		INTO
			:HV-ID
			,:HV-NAME
			,HV-SAL :WS05-NULL-SAL
		FROM 
			EMP
		WHERE ID = :HV-ID
	END-EXEC.
	EVALUATE TRUE
	WHEN SQLCODE = 0
		EVALUATE TRUE
		WHEN WS05-NULL-SAL = 0
		 	DISPLAY ' THE ID    		:' HV-ID
		 	DISPLAY ' THE NAME    	:' HV-NAME-TEXT
		 	DISPLAY ' THE SAL	     		:' HV-SAL
		WHEN OTHER
			DISPLAY ' THE SALARY HAS A NULL VALUE'
		END-EVALUATE
	WHEN SQLCODE = +100
		DISPLAY ' ID    :' HV-ID  ' NOT FOUND'
	WHEN OTHER
		CALL 'DSNTIAR' SQLCA, WS05-ERR-MSG WS05-ERR-LRECL
		DISPLAY ' SELECT FAILED'   
		DISPLAY WS05-ERR-MSG
	END-EVALUATE
	.
HANDLING VARCHAR.

1. FROM THE FILE TO THE DATABASE.
		MOVE THE CONTENT TO THE TEXT PART
		MOVE THE LENGTH OF THE CONTENT TO THE LEN PART
		MENTION THE GROUP ITEM IN THE INSERT STMENT.
2. FROM THE DATABASE TO THE FILE.
		MOVE THE TEXT TO THE OUTPUT LAYOUT.



HANDLING NULL VALUES( NULL INDICATOR)
1. FROM THE FILE TO THE DATABASE.
		* WE MOVE 0 OR -1 TO THE NULL INDICATOR DEPENDING 
				UPON THE EMPTINESS IN THE FILE LAYOUT.
		* WE ATTACH THE NULL INDICATOR IN THE INSERT STATEMENT ALONG WITH 
			HOST VARIABLE.
2. FROM THE TABLE TO THE FILE.
		* THE SYSTEM WILL DEPOSIT 0,-1,-2 INTO THE NULL INDICATOR DEPENDING ON THE 
			VALUE IN THE CORRESPONDING COLUMN.      
		* YOU EVALUATE THE NULL INDICATOR VARIABLE DO PROCESS ACCORDINGLY
